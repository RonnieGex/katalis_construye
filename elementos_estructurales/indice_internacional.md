# Índice

Este índice ahora es interactivo dentro del web book.

Al abrir este documento en la app encontrarás:

- Acordeones por capa: Base, Intermedia y Avanzada.
- Navegación capítulo por capítulo.

Usa la ruta guiada para avanzar en orden o salta al capítulo que necesites hoy.
